# Notifee - Flutter

> ⚠️ **New Architecture Only**: This version of Notifee is built **exclusively for React Native New Architecture** via Flutter's embedding. It requires Flutter 3.22+ with the New Architecture enabled.

A feature rich Android & iOS notifications library for Flutter.

## Platform Requirements

| Requirement | Minimum Version |
|-------------|-----------------|
| Flutter | 3.22+ (New Architecture enabled) |
| iOS Deployment Target | 15.0+ |
| Android minSdk | 23+ |
| Xcode | 16.2+ (for iOS development) |

## Installation

```yaml
dependencies:
  notifee: ^latest
```
